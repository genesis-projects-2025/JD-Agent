# Atomic Capabilities Registry (Skills)

Skills represent reusable programmatic classes and tools that agents invoke.

## 1. LLM Handling
* **Prompt Builder**: Resolves dynamic templating.
* **Streaming Handler**: SSE token streaming.
* **JSON Extractor**: Pulls JSON blocks out of raw AI text.

## 2. Shared Memory
* **Session Memory**: In-flight conversation histories.
* **Long Term Memory**: User database context caching.
* **Working Memory**: In-flight agent task scratchpads.\n
